# Scribble Works Website

A modern, responsive static website built with Bootstrap 5 and Material Design UI for Scribble Works Publishing House.

## Overview

Scribble Works Publishing House is transforming education through accessible and effective tools tailored to the unique needs of African classrooms. This website showcases their innovative products and services.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Bootstrap 5**: Modern UI components and grid system
- **Material Design**: Clean, professional design with Material Design icons and components
- **Smooth Animations**: Engaging scroll animations and hover effects
- **Product Showcase**: Detailed sections for all products including:
  - AssignmentX Platform
  - PrepTime Analytics
  - Assignment Workbook Series (First, Second, and Last Assignment)
  - Transformative Boot Camps

## Products Featured

### Digital Platforms

1. **AssignmentX** - Interactive learning platform for mathematics
   - Website: https://assignmentx.scribbleworks.tech/
2. **PrepTime Analytics** - Google Forms and Sheets add-on for assessment analysis
   - Website: https://preptimeanalytics.com/

### Workbook Series

3. **First Assignment** - Foundation building workbook for Form 1
4. **Second Assignment** - Intermediate level workbook for Form 2
5. **Last Assignment** - Final preparation workbook for Form 3

### Additional Programs

6. **Transformative Boot Camps** - Immersive, hands-on learning experiences

## Customization

### Colors

The color scheme can be customized in `css/style.css` by modifying the CSS variables:

```css
:root {
  --primary-color: #1976d2;
  --secondary-color: #424242;
  --success-color: #388e3c;
  --info-color: #0288d1;
  --warning-color: #f57c00;
  --danger-color: #d32f2f;
  --purple-color: #7b1fa2;
}
```

### Content

Edit the `index.html` file to update text, links, or add new sections.

### Styling

Modify `css/style.css` to change the appearance and animations.

Update `js/script.js` to add or modify interactive features.
